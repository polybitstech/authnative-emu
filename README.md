# authnative-emu

osu! AuthNative.dll emulator based on [osu-kb](https://github.com/polybitstech/osu-kb).

## Caution

1. The data files should be in `%APPDATA%\osu`. Otherwise, it will not work.
2. Do not AFK more than 200 mintues or it might cause kind of UB.
3. **Outdated as 2024-04-19, and not going to update furthermore**

## Build

1. MSVC required. No MinGW.
